# 📖 Overview

Veritas is an AI-powered Fake News Detection System that leverages Machine Learning and Natural Language Processing (NLP) to classify news articles as genuine or potentially misleading.

The system analyzes textual content, extracts meaningful features using NLP techniques, and predicts the credibility of news articles through trained machine learning models.

---

# ✨ Features

✅ Fake News Detection

✅ Natural Language Processing (NLP)

✅ TF-IDF Text Vectorization

✅ Logistic Regression Classification

✅ Naive Bayes Classification

✅ Credibility Analysis

✅ REST API Support

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/prajwal972/veritas-fake-news-detection.git
```

Move into the project folder:

```bash
cd veritas-fake-news-detection
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python src/app.py
```

Open your browser:

```text
http://127.0.0.1:5000
```
